# AGENTS.md

このプロジェクトは、個人用の家計簿・支出メモWebアプリである。

## 目的

- iPhoneのメモ代わりに、支出を素早く記録できるアプリを作る。
- 最初は高機能な家計簿ではなく、入力速度と継続利用を優先する。
- 将来的にはPWA化し、スマホのホーム画面から使える状態を目指す。

## 開発方針

- MVPは小さく保つ。
- まずはローカル保存で十分とする。
- 入力導線は「金額 -> カテゴリ -> メモ任意 -> 保存」を基本にする。
- スマホ幅での使いやすさを優先する。
- 実装時は既存の `README.md` と `docs/requirements.md` を更新しながら進める。

## 初期MVP

- 支出の追加
- 日付、金額、カテゴリ、メモ
- 支出一覧
- 月別合計
- カテゴリ別合計
- ローカル保存

## 後回し

- ユーザー認証
- クラウド同期
- レシートOCR
- 銀行・カード連携
- 複雑な予算管理
